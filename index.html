Tentang Saya

Saya adalah seorang siswa jurusan Rekayasa Perangkat Lunak dari SMK Telkom Lampung. Saat ini, saya sedang mempelajari dasar dari pemerograman web, terutama pada bagian design CSS. Namun, saya lebih tertarik dengan bahasa yang bersifat logika namun simpel secara sintaks, seperti Python. Akan tetapi saya tidak menutup kemungkinan untuk mempelajari bahasa pemerograman lain. Terutama, saat ini studi saya berfokus pada pembelajaran website.

Keahlian

  • Logika Pemerograman: Saya memahami dasar pemerograman seperti variabel, struktur kontrol (perulangan dan percabangan), serta fungsi untuk menyelesaikan masalah teknis.

  • HTML dan CSS Dasar: Untuk saat ini saya masih mempelajari HTML dan CSS. Namun, saya setidaknya memahami bagian dasar dari HTML dan CSS.

  • Python Dasar: Ini adalah bahasa pemerograman yang saya sukai, dan juga saya lebih mendalami bahasa ini. Akan tetapi, saya belum sepenuhnya menguasai bahasa Python, saya baru menguasai atau setidaknya paham dasar-dasarnya.

Proyek

Untuk saat ini saya belum memiliki project nyata yang bersifat komersial atau digunakan orang banyak dalam kehidupan sehari-hari. Meskipun begitu, saya memiliki beberapa project untuk studi atau pembelajaran mandiri saya. Berikut adalah beberapa project saya:

Web Biografi Pahlawan

Merancang dan membangun website biografi responsif dengan menggunakan HTML5 dan CSS3. Proyek ini menerapkan semantic HTML dan teknik Flexbox untuk desain yang modern

Hangman Game

Membangun aplikasi permainan tebak kata interaktif di mana pemain harus menebak kata tersembunyi dengan memberikan tebakan huruf yang benar untuk menggapai kemenangan, kekalahan terjadi ketika kesempatan untuk kesalahan habis.

Flashcard App

Mengembangkan sistem belajar mandiri menggunakan Python dan JSON sebagai penyimpanan data, yang memungkinkan pengguna menambah dan menghapus kartu soal secara dinamis.
